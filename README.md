# Manish Tech Solutions

A modern, responsive business website developed using HTML5, CSS3, and JavaScript.

## 👨‍💻 Founder

**Manish Kumar**

## 📌 Project Overview

Manish Tech Solutions is a responsive business website created as a Web Development Final Project. It demonstrates modern web design, responsive layouts, semantic HTML, CSS styling, JavaScript interactivity, and form validation.

## 🚀 Features

- Fully Responsive Design
- Modern User Interface
- Multi-page Website
- Home Page
- About Page
- Services Page
- Contact Page
- Contact Form Validation
- Dark Mode
- Smooth Animations
- Responsive Navigation
- SEO Friendly HTML
- Clean Folder Structure
